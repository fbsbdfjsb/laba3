**Алгоритм**.

Нашла в википедии историю происхождения слова "Алгоритм":

Алгоритм — это точная последовательность команд, которая от начальных данных приводит к нужному результату за конечное число шагов.

**Блок-схемы**.

В русской Википедии статья "Блок-схема" описывает стандартные обозначения:

овалы для начала/конца, прямоугольники для действий/процессов, ромбы для условий/решений, параллелограммы для ввода/вывода данных, стрелки для потока выполнения.


С помощью интеллектуального помощника perplexity, узнала когда «блок-схемы алгоритмов» стали называть «схемы алгоритмов» и почему:

"Блок-схемы алгоритмов" стали официально называть "схемами алгоритмов" в СССР с 1980 года по ГОСТ 19.002-80 ЕСПД, для унификации с другими схемами (электрическими, логическими) в единой системе; термин "блок-схема" остался в общеупотреблении.
<br>
<br>
**Отечественный стандарт по рисованию схемы алгоритма**

Стандарт — ГОСТ 19.701-90 "ЕСПД. Схемы алгоритмов, программ, данных и систем. Условные обозначения и правила выполнения". В конце — примеры: прямоугольники с закругленными углами для команд, ромбы для ветвлений, стрелки слева/сверху внутрь, справа/снизу наружу.

**Workflow Diagram и связь с Flowchar**

Workflow Diagram (WFD) — диаграмма рабочих процессов, визуализирующая последовательность задач, ролей и решений в бизнесе (с "дорожками" для исполнителей).
WFD — разновидность Flowchart, адаптированная для workflow (потоков работ), с похожими символами (ромбы, прямоугольники), но фокусом на бизнес-процессах и участниках, а не чистых алгоритмах.

**BASE64 + API**

Base64 — это способ закодировать двоичные данные (байты) в обычный текст, чтобы их можно было безопасно передавать там, где разрешены только печатные символы (например, в URL, e‑mail, JSON).

Основная идея Base64
- Данные берутся по 3 байта (24 бита) и делятся на группы по 6 бит, каждые 6 бит превращаются в один символ из специального набора — «алфавита» Base64.  
- Число 64 в названии означает, что алфавит содержит 64 разных символа, то есть это система счисления с основанием 64 (radix‑64).
Алфавит Base64 и почему 64 символа
- Стандартный Base64-алфавит включает:  
  - 26 заглавных букв: A–Z  
  - 26 строчных букв: a–z  
  - 10 цифр: 0–9  
  - 2 спецсимвола: + и /  
  Итого 26 + 26 + 10 + 2 = 64 символа.\
- 64 — это \(2^6\), поэтому 1 символ Base64 кодирует ровно 6 бит, а каждые 4 символа Base64 представляют 24 бита, то есть ровно 3 исходных байта — удобно для передачи байтовыми потоками.

- какой уровень математики и теории нужен;
